GIT STEPS

  1. Create a cloud repository and clone it to your computer
  2. Immediately create a new branch with "git checkout -b branchname"
  3. Once we're done with the changes, perform a "git add -A"
  4. Git status command to check that it was added
  5. Git commit -m with notes on what has been changed
  6. Git push *enter* and we should copy the suggested command because it's a new branch
  7. Create a pull request on Github.com and merge the code
  8. In the terminal, switch to main branch and then git pull so you sync up to that new code
  9. Delete the branch locally and remotely(cloud) to maintain a clean work environment
  10. Repeat steaps 2-now
